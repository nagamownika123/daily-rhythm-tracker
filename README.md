# Daily Rhythm Tracker

A personal static tracker for routine, gym, meals, work blocks, Scooby walk, skin care, fruit, and food-prep reminders.

## GitHub Pages

This folder can be hosted directly from the repository root with GitHub Pages.

1. Create a GitHub repository.
2. Upload these files into the repository root.
3. In GitHub, open **Settings > Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**.
6. Open the published Pages URL on your phone.

## Phone Install

After the GitHub Pages URL opens:

- iPhone: Safari > Share > Add to Home Screen
- Android: Chrome > menu > Add to Home screen

The tracker saves checkmarks in the browser on that device.
